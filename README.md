## Tech Stack

- Framework: Flutter (Dart)
- State Management: Provider
- Networking: HTTP client via custom `ApiService`
- Local Storage: Shared Preferences, Flutter Secure Storage
- Database: SQLite (`sqflite`)
- Location: Geolocator, Geocoding
- Media/File Handling: Image Picker, File Picker
- Permissions: permission_handler
- Sharing & Launching: share_plus, url_launcher
- QR/Barcode: qr_flutter, mobile_scanner, barcode_widget
- Bluetooth Printing: flutter_bluetooth_serial, esc_pos_utils_plus
- UI Utilities: shimmer, intl
- Platforms: Android, Web
